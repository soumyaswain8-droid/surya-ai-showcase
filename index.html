08 · Frontier infrastructure

Strait

“When software starts paying software.”

As AI agents begin doing real work, they'll need to pay each other. Strait is the rails — an agent authorises, Strait routes across the cheapest network and settles in ~200ms with a signed receipt.

Each agent gets a signed identity with per-agent budgets & policies
One call routes across UPI, cards, ACH, SEPA, stablecoins — cheapest wins
Every payment carries a signed, no-chargeback receipt
Built on the India-first “Artha” core — Rust-native, open-spec
